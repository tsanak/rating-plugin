Rater - a simple , lightweight rating plugin

You can use any icon you want

Dependencies

Rater depends only on Jquery .

How to use

    Using the plugin is quite simple
  • just put a class named rater to an input type='number' <input type="number" class='rater'>
  • tempering with the max attribute of the input results in more/less stars <input type="number" min='1' max='10' class='rater'>
  • attribute remove-text determines the value of the text for the element that removes the rating <input type="number" max='5' remove-text='Remove rating' class='rater'>
  • attribute remove-icon determines the value of the icon for the element that removes the rating <input type="number" max='5' remove-icon='glyphicon glyphicon-remove' class='rater'>
  • attribute combination of remove-text & remove-icon <input type="number" max='5' remove-icon='glyphicon glyphicon-remove' class='rater'>
  • attribute icon-full determines which icon will be used on the selected stars,
    attribute icon-empty determines which icon will be used on the unselected stars <input type="number" max='5' class='rater' remove-text='Remove rating' icon-full='fa fa-star' icon-empty='fa fa-star-o'>
  • you can use any combination of custom icons for the attibutes icon-full & icon-empty <input type="number" max='5' class='rater' remove-text='Remove rating' icon-full='glyphicon glyphicon-star' icon-empty='fa fa-star-o'> <input type="number" max='5' class='rater' remove-text='Remove rating' icon-full='glyphicon glyphicon-star' icon-empty='fa fa-star-o'>

Examples

Default icons for stars.

<input type="number" min='1' max='5' class='rater'>

Custom [bootstrap] icons for stars, custom icon for "remove" and remove text.

<input type="number" min='1' max='5' class='rater' remove-text='Remove' remove-icon='glyphicon glyphicon-remove' icon-full='glyphicon glyphicon-star' icon-empty='glyphicon glyphicon-star-empty' >

Custom [fontawesome] icons for stars, custom text for remove.

<input type="number" min='1' max='5' class='rater' remove-text='Remove rating' icon-full='fa fa-star' icon-empty='fa fa-star-o'>

Custom [fontawesome] & [bootstrap] icons for stars.

<input type="number" min='1' max='5' class='rater' icon-full='fa fa-star' icon-empty='glyphicon glyphicon-star-empty'>

Custom [fontawesome] icons for stars, custom icon for remove.

<input type="number" min='1' max='5' class='rater' remove-icon='fa fa-ban' icon-full='fa fa-bell' icon-empty='fa fa-bell-o'>
